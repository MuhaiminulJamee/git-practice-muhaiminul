# Basic Calculator with Name and Date Display

## Project Overview

This project is a simple Python-based application that prints my name along with the current date. It also includes a basic calculator feature that performs fundamental mathematical operations.

The calculator functionality was developed in a separate branch named `feature/calculator` and later merged into the `main` branch.

This project demonstrates:
- Basic Python programming
- Modular function creation
- Git branching and merging
- Clean project structure

## Features

### Main Branch
- Prints my name
- Prints today's current date

### Feature Branch: `feature/calculator`
This branch includes a basic calculator that performs:

- Addition
- Subtraction
- Multiplication
- Division

After implementation, the branch was merged into the `main` branch. Finally, it prints both name & date and performs basic calculator operations.
